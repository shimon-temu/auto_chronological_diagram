入力形式

Auto Chronological Diagramへの入力は画面の「ファイルを選択」から行ってください。

xlsx拡張子のファイルで、以下の形式に従ったものを入力してください。

  • 「ノード情報」「リンク情報」という名前の2つのシートがある
  • 「ノード情報」のA1、B1、C1はそれぞれ「名前」「所属」「重要度」にしてください
  • 「ノード情報」の各行が一人の登場人物を表します
  • 「所属」は「,」で区切ってください(空白はなし）
  • 「ノード情報」シートのC列の重要度は1-5で指定してください
  • 「リンク情報」のA1、B1、C1、D1はそれぞれ「From」「To」「関係性」「双方向」にしてください
  • 「リンク情報」の各行が一つの関係性を表します
  • 「リンク情報」の「From」と「To」は「ノード情報」の「名前」に存在するノードを選んでください
  • 「リンク情報」シートのD列は双方向なら「T」、単方向なら「F」にしてください
